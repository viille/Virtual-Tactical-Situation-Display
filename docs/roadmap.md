# Roadmap

1. Desktop MVP:
- SimConnect adapter lifecycle + reconnect.
- 2D tactical scope with filters, labels, trails.
- local JSON configuration.

2. Stabilization:
- typed managed SimConnect integration with SDK DLL.
- target list panel and sortable metadata.
- better interpolation and prediction.

3. Expansion:
- bullseye and BRAA tools.
- CAP/orbit overlays.
- snapshot/export/replay.

4. Cockpit phase:
- WASM gauge renderer using shared `TacticalDisplay.Core`.
