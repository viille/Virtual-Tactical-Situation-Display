# Symbology

- `OWN`: center triangle with heading line.
- `FRIEND`: circle.
- `PACKAGE`: diamond.
- `SUPPORT`: square.
- `UNKNOWN`: dot.
- `STALE`: dimmed symbol and stale label.

Labels:
- `FULL`: callsign/id, range, relative altitude, heading, closure.
- `MIN`: callsign/id, range, relative altitude.
- `OFF`: no labels.
